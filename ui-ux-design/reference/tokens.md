# Design Tokens — Quanture

## Typography scale

```css
/* Tailwind classes */
text-xs    /* 12px — labels, captions */
text-sm    /* 14px — body small, table cells */
text-base  /* 16px — body default */
text-lg    /* 18px — body large */
text-xl    /* 20px — heading small */
text-2xl   /* 24px — heading medium */
text-3xl   /* 30px — heading large */
text-4xl   /* 36px — hero */
```

## Font weights

```tsx
font-normal   // body text
font-medium   // labels, buttons
font-semibold // subheadings
font-bold     // headings
```

## Shadows

```tsx
shadow-sm    // subtle card lift
shadow       // default card
shadow-md    // modal, dropdown
shadow-lg    // overlay panels
shadow-none  // flat elements
```

## Border radius

```tsx
rounded-sm   // input fields, small buttons
rounded-md   // cards, panels
rounded-lg   // modals
rounded-full // avatars, badges, pills
```

## Z-index scale

```tsx
// Use Tailwind z- classes
z-0   // default
z-10  // sticky headers
z-20  // dropdowns
z-30  // modals backdrop
z-40  // modals
z-50  // toasts, tooltips
```
